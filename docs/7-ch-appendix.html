7.1 Glossary

Table 7.1: Glossary of terms
Term Notation Alias Description
observation \(n\) instance, item, row (of data) A unit being measured across the variables
variable \(p\) feature, column (of data) A measure taken across all observations
data \(X_{n \times p}\) We dicuss complete quantitative matrices
basis \(A_{p \times d}\) Of linear projections, linear component mapping of \(p\) down to \(d\) dimensions
projection \(Y_{n \times d} = X_{n \times p} \cdot A_{p \times d}\) embedding The resulting space the data multiplied by the basis
tour A dynamic linear projection animated over small changes in the projection basis
Intrinsic data dimensionality \(iid\) The minimal dimensionality data can be represented with
Nonlinear model black-box model A model with numerous or interactive terms that cause an interpretabilty issue
